# くろこ

* node.jsで書かれた超簡易IRCログボット+ビューワーです。

# ぷろだくしょん実行

## 前提

* git
* nodejs
* npm
* mongodb

## こまんど

```
git clone https://github.com/geta6/node-kuroko
cd kuroko
npm -g install coffee-script forever
npm install
npm start
```

## ていし・さいきどう

```
npm stop
npm restart
```
