# city

template for node.js

## module

* `less`
* `coffee-script`
* `underscore`
* `jade`

## require for production

* `npm -g install forever`

## initialize

```
npm install
npm -g install forever coffee-script
```

## execute

```
coffee ./application.coffee
```

### production

3 scripts defined.

```
npm start
npm stop
npm restart
```
